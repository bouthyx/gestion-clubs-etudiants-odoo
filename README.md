# Gestion des clubs étudiants – Odoo

## Description
Ce projet consiste à développer un module Odoo permettant la gestion des clubs étudiants.
Il permet de gérer les clubs, les étudiants membres, les responsables de clubs ainsi que les événements organisés.

## Fonctionnalités
- Gestion des clubs étudiants
- Gestion des étudiants et date d’adhésion
- Désignation d’un responsable de club
- Calcul du nombre d’étudiants par club
- Gestion des événements (date, lieu, club)

## Technologies utilisées
- Odoo 17
- Python
- XML
- PostgreSQL
- Docker & Docker Compose
